# 工程优化记录

- 多页面拆分：项目操作、总览、我的项目、详情、事件、奖励中心。
- 自动读取 Ignition 部署地址和 Hardhat ABI，并检查地址字节码。
- 自动检测并切换 Hardhat Local（Chain ID 31337）。
- 使用链上区块时间计算倒计时，支持 `evm_increaseTime` 演示。
- 交易显示钱包确认、广播和区块确认状态，交易期间避免重复提交。
- 事件支持项目过滤、时间、区块号、交易 Hash 和 JSON 导出。
- 里程碑必须在成功前启动，前三阶段按 `goal / 4` 固定释放。
- 失败项目只退未释放资金，动态退款池处理整数除法余数。
- 早期积分采用账户级余额，并通过独立奖励中心兑换纪念凭证。
- 自动化测试覆盖创建、捐赠、结项、提现、退款、里程碑、奖励和取消。
